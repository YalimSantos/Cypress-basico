# Cypress-basico

Projeto desenvolvido no curso de Cypress Básico do professor Walmyr Filho. O site que está sendo testado é o index.html que está na pasta src na raíz do projeto.

## Requerimentos 

Para rodar o projeto, você vai precisar de:
- node
- git

## Instalação

Para fazer as instalações, apenas rode o comando `npm i`

## Rodar os testes

Para rodar os testes no modo headless, apenas use o comando `npm test` ou, para abrir o cypress, use o comando `npm run open`
